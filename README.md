# Encapsulation Basics - Learning Project

A beginner-friendly Java project designed to help you master the core concept of **Encapsulation** in Object-Oriented Programming.

### 🎯 Learning Objectives

By completing this project, you will learn how to:

- Use `private` access modifiers to hide class fields
- Implement proper **Getters** and **Setters**
- Create and use different types of **Constructors** (default, parameterized, and copy)
- Add basic validation logic inside setters
- Write unit tests using JUnit 5
- Understand why encapsulation is important for data protection and maintainability
